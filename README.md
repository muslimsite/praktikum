# Лендинг практикума — Хадиджа умм Ильяс

Одностраничный лендинг практикума «Заготовки без целого дня на кухне».
Статическая реализация финального дизайна (`Лендинг практикума - финал.dc.html`)
из проекта Claude Design — без рантайма Design Composer, полностью
самодостаточная и готовая к публикации.

## Структура

```
index.html              — страница лендинга
assets/css/tokens.css   — токены дизайн-системы (цвета, шрифты, отступы)
assets/css/landing.css   — стили компонентов и секций страницы
```

## Секции

1. **Hero** — заголовок, оффер, кнопка «Хочу на практикум», плашка «1 час».
2. **Проблема** — «Кажется, что заготовки занимают целый день?».
3. **Программа** — 4 карточки: мясные заготовки, завтраки, соусы, секреты.
4. **Калькулятор экономии** — интерактивный: два ползунка (вечеров
   готовки в неделю и минут за раз) считают сэкономленные часы в месяц
   и дни в год в реальном времени.
5. **Цифры** — 1 час · 8 · ~12 · ≈2 недели.
6. **Результат** — чек-лист «Что у вас останется».
7. **Финальный CTA** — переход в Telegram (`@Hadi_UmmIlyas`).

## Интерактив и анимации

- Появление секций по скроллу с каскадом, счётчики цифр, прогресс-бар
  прокрутки, липкая CTA-панель, подъём карточек и зум фото при наведении.
- **Калькулятор экономии времени** (`assets/js/anim.js`) — живой пересчёт.
  Формула прозрачна: готовка «с нуля» каждый вечер против часа заготовок
  на ~12 ужинов + короткий разогрев. Константы вынесены в начало функции
  `setupCalc()` — их легко поправить под реальные цифры практикума.
- Всё сделано как progressive enhancement и уважает
  `prefers-reduced-motion`.

## Дизайн-система

Реализованы токены и компоненты бренда **Хадиджа умм Ильяс**:

- **Цвета** — глубокий сливовый (`--plum-700`) как основная поверхность,
  тёмный баклажан для акцентных блоков, кремовый для светлых секций,
  мягкий зелёный для кнопок, лавандовый и ягодный акценты.
- **Шрифты** (Google Fonts, полная кириллица) — Oswald (заголовки, ALL CAPS,
  широкий трекинг), Golos Text (текст), Caveat (рукописные подписи).
- **Кнопка** — зелёная «пилюля» (primary), точная копия компонента `Button`.
- **Разделитель** — фирменный «потёк джема» (`DripDivider`, seed = 3),
  вшит как статический SVG.

Все снимки сжаты в прогрессивный JPEG (~100–130 КБ). Чтобы заменить фото,
положите новый файл под тем же именем в `assets/img/`.

## Публикация

Это статический сайт — достаточно раздать файлы любым статическим хостингом
(GitHub Pages, Netlify, nginx). Точка входа — `index.html`. Для локального
просмотра можно открыть `index.html` в браузере или поднять
`python3 -m http.server`.
